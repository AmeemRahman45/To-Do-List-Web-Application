<h1>To-Do List (Web Application)</h1>
<p>This project is a simple To-Do List web application built using:</p>
<br>
<ul>- HTML</ul>
<ul>- CSS</ul>
<ul>- JavaScript</ul>
<ul>- Tailwind CSS</ul>
<br>
<p>

  experience.
</p>
